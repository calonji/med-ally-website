# MedAlly Visual System

Use this header for image, video, and 3D prompts.

Palette: clinical white `#FFFFFF`, deep medical navy `#0A2540`, trust teal `#00C2B2`, oxygenated mint `#A6F4E1`, soft slate `#E6EBF1`. Accent: alert coral `#FF6B6B` only for "before MedAlly" pain states.

Typography feel: geometric sans for UI mockups, Inter or Sohne; serif accents only for editorial moments.

Mood: calm, evidence-based, post-burnout. Bright but not toy-like. Confident, not clinical-sterile.

Lighting: soft daylight, low-contrast shadows, no harsh fluorescent ceiling glow.

People: real-feeling clinicians of varied ethnicities and ages, scrubs or business-casual lab coats, no stock-photo "doctor with stethoscope facing camera" cliche.

Anti-patterns to forbid: futuristic blue HUD, matrix code overlay, robot hand touching human hand, glowing brain hologram, stethoscope on laptop. MedAlly is software for grown-up clinicians, not a sci-fi pitch.

Reusable suffix:

Style: editorial product photography, soft natural light, shallow depth of field, color grade favoring teal-and-navy, no HUD overlays, no glowing brain holograms, no stock-photo cliches. Aspect 16:9 unless stated.
